A Chrome extension to temporarily lock access to WhatsApp Web with a custom screen.

## Features
- Lock/unlock WhatsApp Web with a button
- Prevent unauthorized viewing of messages
- Lightweight and easy to use

## How to Use
1. Load the extension in Chrome via `chrome://extensions`.
2. Click "Load unpacked" and select this folder.
3. Click the extension icon to lock/unlock the WhatsApp Web screen.
